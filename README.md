## Usage
### (BEFORE PROCEEDING MAKE SURE YOU HAVE THE [VB-CABLE](https://download.vb-audio.com/Download_CABLE/VBCABLE_Driver_Pack45.zip) DRIVERS INSTALLED)
### (MAKE SURE YOUR 3DS AND PC ARE IN THE SAME NETWORK)

### PC
Install Python (3.12 recommended) and install requirements from requirements.txt

and start server.py

### 3DS
Enter your PC's IP address and proceed.
